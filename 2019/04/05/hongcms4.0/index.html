挺不错的一个CMS，防护做的很好，某些细节上的问题导致了漏洞

背景介绍

HongCMS是一个轻量级的中英文企业网站系统，访问速度极快，使用简单，程序代码简洁严谨，功能强大，完全免费开源，可用于建设各种类型的中英文网站，同时它是一个小型开发框架。
